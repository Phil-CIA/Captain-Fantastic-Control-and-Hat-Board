# Hardware Notes

This folder contains the hardware design files for the Captain Fantastic base control board and its companion HAT board.

## Contents
- `control-board/` – current base control-board KiCad project and related files
- `control-board/archive/` – older control-board snapshots and backup exports kept for reference
- `hat-board/` – companion HAT-board KiCad project

## Guidance
- The HAT board is **not** a revision of the control board; it was created when the original control board was split during development.
- Treat the uploaded KiCad files now stored under `hardware/` as the **authoritative baseline** for the next redesign pass.
- As redesign decisions are made, keep the active source clearly identified and move stale artifacts out of the way.
